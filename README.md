# Fullstack exercise - Articles app

### Features
 * CRUD - articles, comments, users
 * Server Side Rendering - Protected Routes, Better Performance, SEO
 * User Authentication
 * Single or Multi-user Option (private articles)
 * User Dashboard with sortable table
 * Pagination
 * Responsive Design
